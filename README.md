# FSD Sneakers

## Description

A simple React project created with Vite and TypeScript.

## Installation

```sh
pnpm install
```

## Run the Project

```sh
pnpm start
```

## Run JSON Server

```sh
pnpm run server
```

## Run Both (Frontend + Server)

```sh
pnpm run dev
```

## Project Structure

The project follows the Feature-Sliced Design (FSD) architecture.

## Used Technologies

### Main Technologies

- React 19
- TypeScript
- React Router DOM 7
- Vite
- pnpm

### Additional Tools

- JSON Server
